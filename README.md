# JWT authentication system API
this is application for handelling user identity using JWT tokens. it has several features listdown bellow.
what you can do with this application is use the api for firther customization or use entire application as it is for your project.

## how to user this application

## technologies used
- Python
- FastAPI
- MongoDB
- Pydantic
- JWT
- Tailwind CSS
- Nuxtjs
- NuxtUI

### for general users
1. signup users 
    1. with email and password
2. verify signup user email

### for admin users
1. create user roles by admin
2. assign roles to users by admin
3. view all users by admin
4. view users' system histry
